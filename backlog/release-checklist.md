# Release Checklist

## Security
- [ ] API keys only in environment variables
- [ ] CORS policy reviewed
- [ ] Abuse protections enabled
- [ ] Secrets scanned before release

## Reliability
- [ ] Health checks enabled
- [ ] Error monitoring configured
- [ ] Uptime monitoring configured

## Quality
- [ ] CI passing on `main`
- [ ] PR review completed
- [ ] Smoke tests completed

## Commercial
- [ ] Terms and privacy pages published
- [ ] Pricing and membership flows tested
- [ ] Billing and refund policy visible
